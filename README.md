# Tree of Savior Addon

■Marketshowlevel Ex  
とらハム様のMarketshowlevelに色々機能を追加しました。  

■MarketSell Ex  
マーケットの販売登録画面に色々表示します。  

■Autosave money  
チーム倉庫展開時、所持金100,000シルバーを残した金額を自動で入金欄にセットします。

■Check Skillshop  
スキルショップ（バフ屋）のスキルを購入時に確認メッセージを表示します。

■Check Blessedshard  
倉庫を開いたときに「祝福された欠片」を所持している場合、確認メッセージを表示します。それだけ。
